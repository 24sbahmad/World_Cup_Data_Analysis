# Historical World Cup Data Analysis
Inspired by the ongoing 2026 FIFA World Cup and the United States' dominant run through Group D, this project analyzes how the USA and other nations have historically performed in the tournament

### The Data
The [dataset](https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017/data) was published on Kaggle by Mart Jürisoo, which includes:
+ results.csv - Every international football match result (date, teams, scores, tournament, and host)
+ goalscorers.csv - Every goal scored in each match
+ shootouts.csv - The outcome of every penalty shootout
+ former_names.csv - Connecting each country that has changed its name
